# US-35 Onboarding Sessions

## Session cluster 1 — Internal admin and operations

- Zielgruppe: platform admin, tenant admin, dispatcher, accounting, controller/QM
- Dauer: 120 Minuten
- Voraussetzungen:
  - UAT tenant ready
  - accounts from [us-35-uat-account-matrix.md](/home/jey/Projects/SicherPlan/docs/uat/us-35-uat-account-matrix.md)
  - latest migration validation evidence
- Agenda:
  1. role and scope recap
  2. planning to finance workflow walk-through
  3. reporting and export controls
  4. defect capture exercise
- Expected outcome:
  - participants can navigate their surfaces and know where scope stops

## Session cluster 2 — Portal users

- Zielgruppe: customer portal user, subcontractor portal admin/user
- Dauer: 60 Minuten
- Voraussetzungen:
  - portal-linked accounts active
  - released sample data available
- Exercises:
  - open released schedules
  - review/download released docs
  - create scoped watchbook or subcontractor self-service update where allowed
- Expected outcome:
  - users understand what is visible, downloadable, and intentionally hidden

## Session cluster 3 — Employee mobile onboarding

- Zielgruppe: employee app user
- Dauer: 45 Minuten
- Voraussetzungen:
  - mobile build installed
  - active employee link
  - released shift data present
- Exercises:
  - sign in
  - open released schedule and documents
  - create watchbook entry
  - submit time event
- Expected outcome:
  - employees can operate the app without leaving their own scope
