# Sprint Documents

These files are the Codex-facing sprint guides for the 12-sprint SicherPlan delivery plan.
Each file groups the stable story IDs (`US-N`) and task IDs (`US-N-TN`) that will be used in later prompt files.

## How to use these files

- Read `AGENTS.md` first for cross-cutting architecture, data, security, UI, and prompt rules.
- Then open the relevant sprint file.
- Later, create one Codex prompt file per task under `docs/prompts/`, named by task ID, for example `US-7-T1.md`.
