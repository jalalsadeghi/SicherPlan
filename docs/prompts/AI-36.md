# AI-36 — Retrieval Quality Hardening for Multilingual SicherPlan RAG

## Purpose

The assistant is retrieving bad grounding sources for Persian questions. Examples observed by the user:

- "چطور میتونم یک پلن جدید برای مشتری ثبت کنم؟" retrieved `F-02`, `E-01`, `E-01`, `E-01`
- "چطور باید مشتری ثبت کنم؟" retrieved `F-02`, `E-01`, `E-01`, `E-01`
- "چطوری یک قرارداد جدید ثبت کنم؟" retrieved `F-02`, `F-02`, `F-02`

This is not acceptable for RAG. Retrieval must locate contextually correct SicherPlan pages and documentation, especially for Persian and mixed-language questions.

This task improves retrieval quality, multilingual query understanding, and source selection so the OpenAI model receives the right facts.

### Step 3 — Query Expansion

Example:

```text
"چطور باید مشتری ثبت کنم؟"
```

Expanded query should include:

```text

```

Example:

```text

```

Expanded query should include:

```text
customer planning record order project plan P-02 C-01 P-03 planning workspace
```

Example:

```text
"چطوری یک قرارداد جدید ثبت کنم؟"
```

Expanded query should include:

```text
contract document customer document platform services PS-01 customer workspace C-01 order attachments P-02
```

### Step 4 — Improve Retrieval Planning

The retrieval planner must map common intents to likely page IDs:

```text
customer_create:
  likely_page_ids: C-01
  module_keys: customers

customer_plan_create:
  likely_page_ids: C-01, P-02, P-03
  module_keys: customers, planning

order_create:
  likely_page_ids: P-02, C-01
  module_keys: planning, customers

employee_create:
  likely_page_ids: E-01
  module_keys: employees

employee_assign_to_shift:
  likely_page_ids: E-01, P-03, P-04, P-05
  module_keys: employees, planning

contract_or_document_register:
  likely_page_ids: PS-01, C-01, P-02
  module_keys: platform_services, customers, planning
  behavior: ask clarifying question if "contract" could mean customer contract, order attachment, document upload, or commercial agreement
```

Do not let dashboard `F-02` dominate retrieval unless the user actually asks about dashboard.
