# AI-69 — Stage Parity Regression Tests for Assistant Field Questions

## Purpose

Prevent Local/Stage differences in the Assistant scope classifier and field/lookup recognition.

The specific regression to prevent:

```text
Local:  was bedeutet Vertragsreferenz -> correct answer
Stage:  was bedeutet Vertragsreferenz -> out-of-scope refusal
```

This task adds parity tests that simulate a backend-only runtime without frontend source files.
