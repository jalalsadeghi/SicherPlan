# AI-47 — Complete EN/DE Expert Knowledge Pack for Core SicherPlan Workflows

## Purpose

The assistant currently answers well only for employee creation because `E-01` has stronger page-help/action facts. Core workflows like customer registration and product overview remain weak.

This task creates a curated **SicherPlan Expert Knowledge Pack** for English and German. The pack must be used by RAG retrieval as content-bearing context. It is not a bank of final answers.

The goal is to give OpenAI the same level of platform understanding that a human SicherPlan expert would have.

---

## Non-Negotiable Rule

This task must create structured facts, not final answers.

Do not write runtime templates like:

```text
If user asks X, return Y.
```

Instead, create retrievable facts:

```json
{
  "workflow_key": "customer_create",
  "facts": [...],
  "source_basis": [...]
}
```

OpenAI must still generate the final answer.
